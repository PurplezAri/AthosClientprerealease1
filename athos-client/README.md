# Athos Client (Fabric 1.21.5)

A lightweight, **Lunar/Feather-style** client-side mod with:
- Mods menu (Right Shift)
- HUD editor (`)
- HUD modules: FPS, Coords, CPS, Keystrokes, Ping, Clock, Direction, Potions
- Config saved to `config/athosclient.json`
